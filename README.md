
# Groovify

Click here to see [Live]( https://groovifymusic.vercel.app/)


## Run Locally

Clone the project

```bash
 https://github.com/arpitapodder6/Groovify.git
```

Go to the project directory
 
```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
